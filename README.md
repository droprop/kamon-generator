# あなたの紋 — 名前から生まれる紋様ジェネレーター

入力された名前（任意の文字列）から決定論的に生成される、世界にひとつの紋（家紋風オリジナル紋様）を表示するWebページです。

同じ名前を入力すれば、何度試しても必ず同じ紋と名前が生成されます。

## 特徴

- **安全・完全クライアントサイド**
  サーバーへのデータ送信、データベース保存、外部API（LLM等）の呼び出しは一切ありません。入力された名前は完全にブラウザ内（ローカル）でのみ処理されます。
- **決定論的な生成ロジック**
  FNV-1aハッシュおよび疑似乱数アルゴリズム（mulberry32）を用い、入力文字から一意に形状パラメータと和風の紋名を算出します。
- **和のミニマルデザイン**
  明朝体をベースに、墨色と和紙のようなオフホワイトを基調とした、シンプルでモダンなデザインを意識しています。
- **画像の保存と共有**
  生成されたSVGは高解像度（1200px）のPNG画像として保存できます。また、X（旧Twitter）への共有リンクも搭載しています。

## 開発・ビルド手順

ローカル環境で動かす場合の手順です：

```sh
npm install
npm run dev      # 開発サーバー起動
npm test         # テスト実行
npm run build    # 本番ビルド (dist/ フォルダに出力)
```
