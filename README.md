# 🌸 あなたの紋 (Your Kamon) 🌸
### 〜 名前から生まれる、世界にひとつだけのオリジナル紋様ジェネレーター！ 〜

「あなたの名前」を入力するだけで、日本の伝統的な家紋風の可愛い・カッコいい紋様が一瞬で生まれるWebアプリです✨

同じ名前を入力すれば、いつでも何回でも必ず同じデザインが再現されます！

## 🚀 このアプリのすてきなところ！

- **🔒 プライバシーは完全セーフ！**
  名前や生成された画像がサーバーに送られることは一切ありません！すべてあなたのスマホやPCのブラウザの中で安全に作られます。
- **🎉 決定論的で面白い！**
  内部でハッシュ計算と疑似乱数アルゴリズム（FNV-1a / mulberry32）を使って、入力された文字から世界にひとつのデザインを導き出します。
- **🍵 和テイスト＆ミニマルモダン**
  明朝体をベースに、墨色と和紙のようなオフホワイトの、とっても美しくて洗練されたデザインです。
- **💾 画像の保存＆SNSシェアもかんたん！**
  できた紋様は1200pxの高解像度PNGでダウンロード可能！X（Twitter）でみんなにシェアしちゃおう！

## 🛠️ ローカルで動かしてみたい人へ

もし自分のパソコンで改造したり遊んだりしてみたいときは、以下のコマンドを使ってね！

```sh
npm install
npm run dev      # 開発サーバーであそぶ！
npm test         # テストを実行する！
npm run build    # 本番用にビルドする！
```
